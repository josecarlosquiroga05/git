  1. ¿Que es un sistema de control de versiones?

    Es un sistema que permite a los desarrolladores gestionar los cambios en el código fuente a lo largo del tiempo, facilitando la colaboración y el seguimiento de las modificaciones realizadas por diferentes miembros del equipo.

  2. ¿Cual es la diferencia entre git y github?

    La diferencia es que git es un sistema de control de versiones y github es una plataforma de alojamiento de código que utiliza git como sistema de control de versiones.

  3. ¿Cómo creo una rama en git?

    Mediante el comando git branch (nombre de la rama)

  4. ¿Cómo clono un repositorio?

    mediante el comando git clone (url del repositorio)

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    mediante el comando git pull origin (nombre de la rama)

  6. ¿Cómo puedo gener un tag?

    mediante el comando git tag (nombre del tag)
